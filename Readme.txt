# Rental Car Website

A sleek, static, and user-friendly web application designed for a premium Car Rental Service. The website features an intuitive layout that allows users to seamlessly browse available vehicles, view rental rates, and find contact details to finalize their bookings.

## 🚀 Features

* **Intuitive & Responsive Layout:** Built with a mobile-first approach, ensuring the site looks great on desktops, tablets, and smartphones.
* **Interactive Vehicle Slider:** Smooth JavaScript-powered image slider/carousel to easily browse through the available fleet.
* **Vehicle Fleet Showcase:** Dedicated sections to view different types of available cars and rental rates.
* **Seamless Inquiry Flow:** Easily accessible contact information to connect users directly with the service for bookings.

---

## 🛠️ Tech Stack

* **HTML5:** Semantic markup for structured website content.
* **Bootstrap:** Used for rapid, responsive grid layouts and modern UI components.
* **CSS3:** Custom styles to enhance the theme and visual appeal.
* **JavaScript:** Handled dynamic UI interactions, specifically the vehicle sliding functionality.
